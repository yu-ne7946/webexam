강아지 왕국 리뉴얼 홈페이지

기존에 있던 '강아지왕국' 홈페이지를 사용자 편의에 맞춰 좀 더 직관적이고 단순하게 만들었습니다. 또한, 관리자 페이지를 만들어 firebase, ajax/json방식으로 데이터를 불러올 수 있게 했습니다.

기존페이지 방문

Ajax,Firebase 를 활용한 navigation

Navigation 중 사료,간식,영양제는 배열을 이용해서 목욕/미용, 배변/위생 json 파일을 만들어서 ajax로 가져오는 방식으로 식기/집, 패션/외출,장난감은 firebase 의 database를 이용해 만들었습니다.
